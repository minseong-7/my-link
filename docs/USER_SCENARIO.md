# 마이링크 (MyLink) 사용자 시나리오

서비스 이용 과정에서 발생할 수 있는 주요 시나리오를 '소유자(크리에이터)'와 '방문자(일반 유저)'의 관점으로 나누어 정리한 문서입니다.

## 1. 소유자 (프로필 및 링크 관리자)

**접근 및 인증**
- 사용자는 [자신의 다중 링크 프로필을 관리]하기 위해 [서비스 랜딩 페이지에서 '구글 소셜 로그인' 버튼을 클릭하여 회원가입 및 로그인]한다.
- 사용자는 [작업을 마치고 계정을 안전하게 보호]하기 위해 [대시보드 상단의 '로그아웃' 버튼을 클릭하여 로그아웃]한다.

**프로필 관리**
- 사용자는 [방문자에게 보여질 자신의 이름, 디스플레이 네임, 소개글을 수정]하기 위해 [대시보드의 프로필 영역 텍스트를 클릭하여 인라인 수정 모드로 진입한 후 값을 변경하고 Enter 키를 눌러 저장]한다.
- 사용자는 [자신의 프로필 페이지 고유 주소를 확인하거나 타인에게 공유]하기 위해 [화면에 표시된 퍼블릭 URL(`url.com/displayName`)을 복사]한다.

**링크 관리**
- 사용자는 [새로운 작업물이나 외부 웹사이트를 등록]하기 위해 [대시보드에서 '링크 추가' 버튼을 클릭]한다.
- 사용자는 [새로 추가한 빈 링크의 내용(제목, URL)을 입력하거나, 기존 링크의 오타 및 URL을 수정]하기 위해 [해당 텍스트를 직접 클릭해 인라인 수정 모드로 변경한 뒤, 내용을 입력하고 Enter 키를 눌러 저장]한다.
- 사용자는 [더 이상 필요 없는 링크를 목록에서 지우기] 위해 [해당 링크 우측에 위치한 삭제 버튼을 클릭하고, 이어서 노출되는 '확인(Confirm) 팝업창'에서 승인하여 삭제를 완료]한다.

---

## 2. 방문자 (퍼블릭 페이지 열람자)

**열람 및 상호작용**
- 사용자는 [특정 크리에이터의 링크 모음을 확인]하기 위해 [제공받은 고유 URL(`url.com/displayName`)을 브라우저에 입력하여 방문]한다.
- 사용자는 [크리에이터가 등록한 외부 사이트나 작업물로 이동]하기 위해 [관심 있는 특정 링크 항목을 클릭]한다. (※ 이 때 시스템은 클릭 수를 카운트하여 통계 데이터로 활용한다)
- 사용자는 [자신만의 다중 링크 프로필 페이지를 개설]하기 위해 [방문한 페이지의 링크 목록 최하단에 위치한 '내 MyLink 만들기' 유도 버튼을 클릭하여 서비스 메인 랜딩 페이지로 이동]한다.
